URL Help

At this time, the format for specifying a URL is:

  /js/?url=http://example.com/path/to/1245.json

Also, only requests to la.indymedia.org are allowed.

Examples: 

 /js/?url=http://la.indymedia.org/news/2013/05/260009.json
 /js/?url=http://la.indymedia.org/news/2013/05/260000.json
 /js/?url=http://la.indymedia.org/news/2013/05/259869.json
 /js/?url=http://la.indymedia.org/news/2013/05/259788.json
